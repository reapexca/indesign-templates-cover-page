>>About the file<<

The file has 4K resolution, which is 3840x2160 px. MockUp is in * .psd format, so in order to open it you need Adobe Photoshop software (you can download it here: http://www.adobe.com/downloads.html). PSD file is well organized, layers and color groups are appropriately named.

>>How to edit a MockUp<<

In order to change the design on the MockUp, you need double-click the left mouse button on the miniature of the smart object, add the template to the opened window and close it herewith saving the changes.
